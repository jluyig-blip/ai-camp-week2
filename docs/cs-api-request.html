API 1 — 1순위 미답변 문의 목록 조회

POST/api/cs/qna_list.php

자사몰·스마트스토어·네이버톡톡·카카오상담톡·지그재그·에이블리 전 채널의 미답변 문의를 가져옵니다.

요청 파라미터

파라미터필수설명예시
status선택미답변(기본값: unanswered) / 전체(all) / 답변완료(answered)unanswered
channel선택채널 필터. 생략 시 전체 채널지그재그
limit선택건수 (기본 50, 최대 200)100
offset선택페이징 오프셋0
date_from선택시작일 YYYY-MM-DD2026-05-24
date_to선택종료일 YYYY-MM-DD2026-05-31

API 2 — 2순위 AI 답변 전송

POST/api/cs/qna_reply.php

AI가 생성한 답변을 WMS에 등록합니다. auto_send=0이면 저장만, auto_send=1이면 고객에게 즉시 발송합니다.

요청 파라미터

파라미터필수설명예시
qna_idx필수답변할 문의 번호 (qna_list 응답의 qna_idx)1877862
reply필수답변 본문 전체안녕하세요, ATTRANGS 입니다~♡ ...
auto_send선택0 = 저장만(기본), 1 = 즉시 발송0
ai_generated선택AI 생성 여부 표시 (감사 추적용) 1/01
model_id선택사용 AI 모델명 (로그용)claude-sonnet-4-6

응답 예시 — 저장 성공

{
  "ok":      true,
  "qna_idx": 1877862,
  "status":  "saved",     // saved | sent
  "sent_at": null        // auto_send=1 이면 발송 시각
}

응답 예시 — 이미 답변됨 (오류)

{
  "ok":   false,
  "code": "already_answered",
  "msg":  "이미 답변이 등록된 문의입니다"
}

⚡ 우선순위 및 일정

순서API이유
1순위 qna_list.php 없으면 미답변 문의를 자동으로 가져올 수 없어 전체 자동화 불가
2순위 qna_reply.php 없으면 AI 답변을 수동 복붙해야 함 (작업 시간 단축 불가)
연동 방법: 구현 완료 후 서버 환경변수 2줄만 추가하면 즉시 연동됩니다. 별도 코드 수정 불필요.
# .env 에 추가
WMS_QNA_URL=https://attrangs.co.kr/api/cs/qna_list.php
WMS_QNA_REPLY_URL=https://attrangs.co.kr/api/cs/qna_reply.php
