## 문의 #011 — 배송 현황 + 선출고 요청 (할당완료 + 지연 혼재)

**채널:** 자사몰
**유형:** 배송문의 / 선출고요청
**긴급도:** 높음

```
제목: 일부 상품 먼저 보내주실 수 있나요?

주문번호: 718****24
성함: 고객님
연락처: 010-****-****

지난주에 티셔츠랑 가디건 같이 주문했는데
티셔츠는 할당됐다고 문자 왔는데 가디건은 아직도 입고가 안 됐다네요.
티셔츠만 먼저 받을 수 있을까요?
가디건은 얼마나 더 기다려야 하는지도 알려주세요.
```

**처리 흐름 (확정 로직 기준)**
1. `order_status.php` 조회 → 품목별 상태 분리
2. 티셔츠: `item_status=할당완료`, `delayed=false` → 마감 시간(자사몰 22:30) 기준 금일/내일 출고 안내
3. 가디건: `delayed=true` → `delay_until` 날짜 안내
4. 선출고 요청 처리:
   - 지연 4일 이상 AND 선출고 총액 30,000원 이상 → 긴급체크 API 호출
   - 조건 미충족 → "선출고 배송비 3,500원 선입금 후 처리 가능" 안내
