## 답변 #001 — 배송 지연 항의 (문의 #001 대응)

**AI 처리:** WMS 조회 → 배송 가능 여부 확인 → 입금 안내 자동 생성  
**사람 검토 필요:** 입금 확인 후 최종 출고 승인

```
안녕하세요, ATTRANGS 입니다~♡

주문번호 [718****01] 확인되었습니다.

입고 지연으로 불편을 드려 대단히 죄송합니다.
추가 배송비 관련 안내드립니다.

[입금 정보]
- 금액: 980원
- 계좌: 국민은행 468001-01-****** / 에스엔패션그룹
- 입금 후 게시판 또는 톡상담에 아래 정보 남겨주세요:
  · 입금 날짜
  · 입금자명
  · 은행명

입금 확인 즉시 당일 출고 진행해드리겠습니다.

다른 문의사항은 아뜨랑스 게시판으로 남겨주시면 빠르게 답변드릴게요.
감사합니다, 행복한 하루 되세요~♡

[담당자 메모]
- 고객 불만 높음 (입고지연 + 추가배송비 이중 불만)
- 입금 확인 후 당일 출고 우선 처리 필요
- 필요시 사과 쿠폰 발급 검토
```

## 답변 #002 — 반품 요청 + 환불 방법 (문의 #002 대응)

**AI 처리:** 주문 확인 → 반품 자동 접수 → 환불 방식 계산  
**사람 검토 필요:** 반품 접수 최종 확인

```
안녕하세요, ATTRANGS 입니다~♡

주문번호 [718****44] 확인되었습니다.

반품 접수 도와드리겠습니다.

[반품 안내]
- 고객님께서 별도로 택배사 접수 안하셔도 됩니다
- CJ대한통운 회수 접수 진행됩니다
- 접수 후 익일부터 1~3일 내 기사님 방문 예정입니다

[환불 안내]
- 반품비(편도 3,000원)는 상품 수거 후 결제금액에서 차감됩니다
- 잔여 금액은 카드 결제 취소로 환불됩니다
- 카드사별로 영업일 3~5일 소요됩니다

다른 문의사항은 아뜨랑스 게시판으로 남겨주시면 빠르게 답변드릴게요.
감사합니다, 행복한 하루 되세요~♡

[담당자 메모]
- 반품 접수 완료 처리
- 환불금액 = 결제금액 - 반품비 3,000원
- 쿠폰 사용 여부 확인 필요
```

```
빠르고 친절하게 답변드릴게요.

```

## 답변 #005 — 주문 취소 요청 (문의 #005 대응)

**AI 처리:** 주문 상태 조회(OMS) → 출고 전 취소 가능 확인 → 취소 처리 + 포인트 복원 안내  
**사람 검토 필요:** 없음 (출고 전 단순 취소)

```
안녕하세요, ATTRANGS 입니다~♡

주문번호 [718****27] 확인되었습니다.

현재 출고 전 상태로 취소 처리 도와드리겠습니다.

[취소 안내]
- 결제 취소: 카드 결제분은 카드사 기준 영업일 3~5일 내 취소됩니다
- 포인트: 결제에 사용하신 포인트는 취소 즉시 전액 복원됩니다

다른 사이즈로 재주문을 원하시면 포인트 복원 확인 후 재주문해주세요.

다른 문의사항은 아뜨랑스 게시판으로 남겨주시면 빠르게 답변드릴게요.
감사합니다, 행복한 하루 되세요~♡

[담당자 메모]
- OMS 취소 처리 완료
- 포인트 복원 완료 확인
- 재주문 안내 완료
```
