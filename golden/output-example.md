## 답변 #001 — 배송 지연 항의 (문의 #001 대응)

**사람 검토 필요:** 입금 확인 후 최종 출고 승인

```
안녕하세요, ATTRANGS 입니다~♡

주문번호 [718****01] 확인되었습니다.

입고 지연으로 불편을 드려 대단히 죄송합니다.
추가 배송비 관련 안내드립니다.

[입금 정보]
- 금액: 980원
- 계좌: 국민은행 468001-01-****** / 에스엔패션그룹
- 입금 후 게시판 또는 톡상담에 아래 정보 남겨주세요:
  · 입금 날짜
  · 입금자명
  · 은행명

입금 확인 즉시 당일 출고 진행해드리겠습니다.

다른 문의사항은 아뜨랑스 게시판으로 남겨주시면 빠르게 답변드릴게요.
감사합니다, 행복한 하루 되세요~♡

[담당자 메모]
- 고객 불만 높음 (입고지연 + 추가배송비 이중 불만)
- 입금 확인 후 당일 출고 우선 처리 필요
- 필요시 사과 쿠폰 발급 검토
```

**AI 처리:** 주문 확인 → 반품 자동 접수 → 환불 방식 계산  
**사람 검토 필요:** 반품 접수 최종 확인

```
안녕하세요, ATTRANGS 입니다~♡

주문번호 [718****44] 확인되었습니다.

[반품 안내]
- 고객님께서 별도로 택배사 접수 안하셔도 됩니다

다른 문의사항은 아뜨랑스 게시판으로 남겨주시면 빠르게 답변드릴게요.
감사합니다, 행복한 하루 되세요~♡

[담당자 메모]
- 반품 접수 완료 처리
- 환불금액 = 결제금액 - 반품비 3,000원
- 쿠폰 사용 여부 확인 필요
```

## 답변 #007 — 사이즈 추천 요청 (문의 #007 대응)

**AI 처리:** 사이즈 추천 기준표 조회 → 키/체중/체형 입력값 매칭 → 사이즈 추천
**사람 검토 필요:** 없음 (기준표 범위 내 단순 추천)

```
안녕하세요, ATTRANGS 입니다~♡

ps6024 오버핏 셔츠 문의 주셨군요!
168cm, 58kg, 어깨 넓은 편 기준으로 안내드릴게요 :)

[사이즈 추천]
→ M 사이즈 추천드립니다

[이유]
- 오버핏 셔츠는 기본적으로 넉넉한 핏으로 나오는 제품입니다
- 168cm / 58kg 체형에 M이 적당한 오버핏으로 연출됩니다
- 어깨가 넓으신 경우에도 M의 어깨 너비(약 48cm)가 충분히 커버됩니다
- L은 전체적으로 너무 크게 입히는 느낌이 날 수 있어요

[참고]
- 더 루즈한 핏을 선호하신다면 L도 괜찮습니다
- 실측 사이즈표는 상품 상세페이지에서 확인 가능합니다

추가 문의 있으시면 편하게 남겨주세요!
감사합니다, 행복한 하루 되세요~♡

[담당자 메모]
- 사이즈 기준표 자동 조회 완료
- 추가 실측 문의 시 상품 상세 안내
```

## 답변 #009 — 교환 접수 요청 (문의 #009 대응)

**AI 처리:** 교환 가능 기간 확인 → 재고 조회(WMS, 화이트 컬러) → 교환 접수 처리
**사람 검토 필요:** 교환 접수 최종 확인

```
안녕하세요, ATTRANGS 입니다~♡

주문번호 [718****55] 확인되었습니다.

색상이 사진과 달라 불편 드려 정말 죄송합니다.
화이트 컬러로 교환 도와드리겠습니다 :)

[교환 안내]
- 교환 상품: 동일 상품 화이트 컬러 ✅ 재고 확인 완료
- 수거 방법: CJ대한통운 회수 접수 진행됩니다 (별도 택배 접수 불필요)
- 접수 후 익일부터 1~3일 내 기사님 방문 예정입니다

[배송비 안내]
- 색상 차이로 인한 교환이므로 왕복 배송비는 고객 부담입니다 (편도 3,000원 × 2)
- 수거 시 동봉 또는 계좌 안내드릴게요

교환 상품은 수거 확인 후 바로 발송해드리겠습니다.
감사합니다, 행복한 하루 되세요~♡

[담당자 메모]
- WMS 재고 조회: 화이트 컬러 재고 있음
- 교환 접수 완료 처리
- 색상 차이 = 단순변심 준용 → 왕복 배송비 고객 부담
- 수거 후 즉시 교환품 발송
```

## 답변 #010 — 불량 교환 요청 (문의 #010 대응)

**AI 처리:** 불량 접수 분류 → 교환 가능 여부 확인 → 왕복 배송비 무료 처리
**사람 검토 필요:** 불량 사진 확인 후 최종 승인

```
안녕하세요, ATTRANGS 입니다~♡

주문번호 [718****39] 확인되었습니다.

새 제품으로 교환 도와드리겠습니다.

[불량 사진 접수]
- 사진은 이 톡상담 채팅창에 바로 올려주시면 됩니다
- 올 나감·구멍 부위가 잘 보이도록 찍어주세요

[교환 안내]
- 불량으로 인한 교환이므로 왕복 배송비 전액 무료입니다 ✅
- 사진 확인 후 CJ대한통운 회수 접수 진행해드립니다
- 접수 후 익일부터 1~3일 내 기사님 방문 예정입니다
- 수거 확인 즉시 새 제품 발송해드리겠습니다

불편 드려 다시 한번 진심으로 사과드립니다.
감사합니다, 행복한 하루 되세요~♡

[담당자 메모]
- 불량 접수: 올 나감 + 구멍 (착용 전 발생)
- 불량 사진 수신 후 최종 교환 승인 필요
- 왕복 배송비 무료 처리 완료
- 재고 확인: 동일 상품 재고 여부 WMS 조회 필요
- 반복 불량 시 품질팀 보고
```
