# 골든 출력 예시

## 문의 #001 — 스마트스토어 / 배송 지연

```
[채널] 스마트스토어
[유형] 배송 지연
[긴급도] 높음

[1차 답변 초안]
안녕하세요, 고객님. 문의 주셔서 감사합니다.

주문번호 12345의 배송이 지연되어 불편을 드려 진심으로 사과드립니다.
현재 배송 상태를 확인하고 있으며, 빠른 시일 내에 정확한 배송 일정을 안내해 드리겠습니다.

잠시만 기다려 주시면 곧 연락드리겠습니다. 감사합니다.

[담당자 메모]
- 택배사 송장 조회 후 현재 위치 직접 확인 필요
- 5일 이상 미배송 → 배송 지연 보상 정책 적용 여부 확인
- 분실 가능성 있을 경우 팀장 보고 후 재발송 or 환불 안내
```

## 문의 #002 — 카카오 / 오배송 + 환불 요청

```
[채널] 카카오
[유형] 오배송 + 환불 요청
[긴급도] 높음

[1차 답변 초안]
안녕하세요 고객님! 불편을 드려서 정말 죄송해요 😥

주문하신 상품과 다른 제품이 배송된 것 확인했어요.
오배송 건은 저희가 빠르게 처리해 드릴게요!

정확한 처리를 위해 아래 정보를 보내주시겠어요?
1. 받으신 상품 사진 1장
2. 주문번호 (주문확인 문자에서 확인 가능)

확인 후 교환 or 환불 중 원하시는 방법으로 안내드릴게요. 감사합니다 🙏

[담당자 메모]
- 오배송 확인 후 회수 택배 예약 필요 (물류팀 연락)
- 재고 있으면 교환, 없으면 환불로 안내
- 환불 승인은 팀장 결재 필요
- 카카오 말투: 친근하게, 이모지 1~2개 허용
```
