# Golden Output Examples

## 출력 1 — 주문취소 (포인트 혼합결제) ← 입력 1 대응

```
[채널] 자사몰  [유형] 주문취소  [긴급도] 중간

[OMS 조회 결과]
주문번호: 718****27 | 상태: 결제완료 (출고 전)
결제수단: 카드 + 포인트 혼합결제
포인트 사용액: 2,000P

[1차 답변 초안]
안녕하세요, ATTRANGS 입니다~♡

주문번호 [718****27] 확인되었습니다.
현재 출고 전 상태로 취소 처리 도와드리겠습니다.

[취소 안내]
- 결제 취소: 카드 결제분은 카드사 기준 영업일 3~5일 내 취소됩니다
- 포인트: 결제에 사용하신 포인트는 취소 즉시 전액 복원됩니다

다른 사이즈로 재주문을 원하시면 포인트 복원 확인 후 재주문해주세요.
감사합니다, 행복한 하루 되세요~♡

[담당자 메모]
- OMS 취소 처리 완료
- 포인트 복원 완료 확인
```

## 출력 5 — 배송일 변경 가능 여부 ← 입력 5 대응

```
[채널] 자사몰  [유형] 배송일 변경문의  [긴급도] 높음

[OMS 조회 결과]
주문번호: 미제공 | 상품명: 차콜 긴팔
예약 배송 예정일: 2026-06-15
현재 상태: 결제완료 (출고 전 — 예약 발송 대기)

[1차 답변 초안]
안녕하세요, ATTRANGS 입니다~♡

차콜 긴팔 주문 건 확인되었습니다.
현재 예약 배송 상품으로 6월 15일 출고 예정이며,
안타깝게도 5월 30일 이전 조기 배송은 어렵습니다.

[안내]
- 조기 수령이 꼭 필요하신 경우, 현 주문 취소 후 일반 재고 상품으로 재주문해 주시면 됩니다.
- 취소를 원하시면 주문번호를 알려주시면 바로 처리 도와드리겠습니다.

불편을 드려 죄송합니다, 행복한 하루 되세요~♡

[담당자 메모]
- 예약 상품 특성상 배송일 임의 변경 불가
- 고객이 취소 원할 경우 OMS 즉시 처리
```

## 출력 6 — 사이즈 교환 ← 입력 6 대응

```
[채널] 자사몰  [유형] 사이즈 교환  [긴급도] 중간

[OMS 조회 결과]
주문번호: 71981624 | 상태: 배송완료
수령일: 조회 필요 | 교환 가능 기간: 수령 후 7일 이내

[1차 답변 초안]
안녕하세요, ATTRANGS 입니다~♡

주문번호 [71981624] 확인되었습니다.
M 사이즈 → L 사이즈 교환 요청 도와드리겠습니다.

[교환 안내]
- 교환은 상품 수령 후 7일 이내, 미착용·태그 부착 상태에서 가능합니다.
- L 사이즈 재고 확인 후 교환 진행되며, 재고 부족 시 환불로 안내드립니다.
- 교환 배송비: 왕복 5,000원 (제품 불량 시 무료)

교환 신청을 진행해 드릴까요? 확인 후 바로 처리 도와드리겠습니다~♡

[담당자 메모]
- L 사이즈 재고 확인 필수
- 수령일 기준 7일 초과 여부 OMS에서 재확인
```

## 출력 7 — 출고 지연 ← 입력 7 대응

```
[채널] 자사몰  [유형] 출고 지연  [긴급도] 높음

[WMS 조회 결과]
주문번호: 미제공 | 주문일: 약 5일 전
현재 상태: 결제완료 (출고 대기 중)
출고 예정: WMS 확인 필요

[1차 답변 초안]
안녕하세요, ATTRANGS 입니다~♡

주문하신 지 5일이 경과했음에도 출고가 지연되어 불편을 드려 정말 죄송합니다.

[현재 상황]
- 주문번호를 알려주시면 해당 건을 즉시 확인하여 출고 처리 도와드리겠습니다.
- 현재 물류 처리 지연이 발생한 경우, 우선순위로 처리해 드리겠습니다.

빠른 시일 내 출고될 수 있도록 최선을 다하겠습니다~♡

[담당자 메모]
- 주문번호 확보 후 WMS에서 출고 지연 원인 확인 (재고 부족 / 물류 적체 / 시스템 오류)
- 5일 이상 지연 시 사과 쿠폰 발급 여부 매니저 확인
```

---

## 출력 포맷 규칙 (3주차 제약 조건 반영)

1. **OMS/WMS 조회 결과 섹션 필수** — 배송·취소 문의 시 `[OMS 조회 결과]` 또는 `[WMS 조회 결과]` 블록을 답변 앞에 출력. 조회 실패 시: `"WMS 조회 실패 — 수동 확인 필요"` 명시.
2. **날짜·상태 추측 금지** — 조회 결과 없으면 날짜·배송 상태 절대 추측 안 함.
3. **채널 밖 접촉 약속 금지** — "전화 드리겠습니다" 등 채널 외부 접촉 약속 자동 제거.
